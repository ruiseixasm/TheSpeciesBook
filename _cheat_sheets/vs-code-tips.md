VS Code + LaTeX Workshop Shortcuts

Essential VS Code Shortcuts:

Ctrl+P - Quick file open
Ctrl+Shift+P - Command palette
Ctrl+Shift+E - File explorer
Ctrl+Shift+G - Source control (Git)
Ctrl+` - Toggle terminal
Ctrl+, - Settings

Editing Shortcuts:

Ctrl+C / Ctrl+V - Copy/paste
Ctrl+X - Cut line
Ctrl+Shift+K - Delete line
Alt+Up / Alt+Down - Move line up/down
Ctrl+Z / Ctrl+Y - Undo/redo
Ctrl+F / Ctrl+H - Find/replace
Ctrl+D - Select next occurrence
Ctrl+Shift+L - Select all occurrences

LaTeX Workshop Shortcuts:

Ctrl+Alt+B - Build LaTeX project
Ctrl+Alt+V - View PDF
Ctrl+Alt+J - Sync PDF to cursor
Ctrl+Alt+E - View LaTeX errors
Ctrl+Space - Trigger autocomplete

Useful Features:

Structure View - Click outline icon in top-left
Problems Tab - Ctrl+Shift+M shows errors
PDF Viewer - Opens side-by-side with Ctrl+Alt+V

Git Integration:

Source Control tab (Ctrl+Shift+G)
Click + to stage files
Type message and press Ctrl+Enter to commit
Click ... for push, pull, branch
