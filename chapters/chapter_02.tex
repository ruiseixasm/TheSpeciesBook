\chapter{Premise}

The Cambrian Explosion is a fascinating event in the evolution  of life, this
event is so astonishing that romanticism finds roots to feed in and give some
comfortable benign interpretation, like a creator that designed the existing organisms.

The premise in this book is not that different, except that is not the benign explanation
nor as divine as it normally people want.
